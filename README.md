# Goat Store UI Vue

A Vue front to the [goat-store api](https://github.com/euriphides/goat-store) 

## Demo

https://euriphides.github.io/goat-store-ui-vue/

## Install

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build and Deploy

Make sure your git working directory is clean, and everything is checked in, then:
```bash
npm run build
npm run deploy
```
